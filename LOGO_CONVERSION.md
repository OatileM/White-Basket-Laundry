# Logo Conversion Instructions

## Required Files to Create

### 1. logo.png
Convert the Logo.pdf to PNG format with these specifications:
- **Dimensions**: 200px width × 60px height (maintain aspect ratio)
- **Format**: PNG with transparent background
- **Quality**: High resolution for crisp display on all devices
- **File size**: Optimize to under 50KB for fast loading

### 2. favicon.ico
Create a favicon from the logo with these specifications:
- **Dimensions**: Multiple sizes in one file (16×16, 32×32, 48×48)
- **Format**: ICO format for best browser compatibility
- **Background**: Transparent or white
- **File size**: Under 10KB

## Conversion Tools

### Online Tools (Free)
1. **CloudConvert** (cloudconvert.com)
   - Upload Logo.pdf
   - Convert to PNG
   - Resize to 200×60px

2. **Favicon.io** (favicon.io)
   - Upload the converted PNG
   - Generate favicon.ico automatically

### Professional Tools
1. **Adobe Illustrator/Photoshop**
   - Open Logo.pdf
   - Export as PNG (200×60px)
   - Save for Web (optimized)

2. **GIMP** (Free alternative)
   - Import Logo.pdf
   - Scale image to 200×60px
   - Export as PNG

## File Placement
Once created, place these files in the root directory:
```
White Basket Laundry/
├── logo.png          # Main logo for website
├── favicon.ico       # Browser tab icon
└── ...other files
```

## Testing
After adding the files:
1. Open index.html in a browser
2. Verify logo displays correctly
3. Check favicon appears in browser tab
4. Test on mobile devices for clarity

## Backup
Keep the original Logo.pdf file as the master copy for future use.
