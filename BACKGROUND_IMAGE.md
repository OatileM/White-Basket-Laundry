# Background Image Instructions

## Required Background Image

### File Specifications
- **Filename**: `hero-bg.jpg`
- **Dimensions**: 1920px × 1080px (Full HD)
- **Format**: JPG (optimized for web)
- **File size**: Under 500KB for fast loading
- **Quality**: High resolution but web-optimized

### Image Content Recommendations
Perfect images would show:
- **Modern laundromat interior** with washing machines and dryers
- **People using the facilities** (diverse, professional-looking customers)
- **Clean, bright environment** with good lighting
- **Professional staff** helping customers (optional)
- **Folded clothes or laundry baskets** in the scene

### Image Style
- **Bright and welcoming** atmosphere
- **Professional quality** photography
- **Good contrast** (darker areas for text overlay)
- **Horizontal orientation** (landscape)

### Where to Find Images

#### Stock Photo Sites (Paid - High Quality)
1. **Shutterstock** - Search: "modern laundromat people"
2. **Getty Images** - Search: "laundry service customers"
3. **Adobe Stock** - Search: "commercial laundromat interior"

#### Free Stock Photo Sites
1. **Unsplash** (unsplash.com) - Search: "laundromat" or "laundry"
2. **Pexels** (pexels.com) - Search: "washing machine people"
3. **Pixabay** (pixabay.com) - Search: "laundromat interior"

#### Custom Photography
- Hire a local photographer to take pictures of your actual facility
- Include real customers (with permission)
- Show your unique branding and atmosphere

### Image Placement
Place the image file in the root directory:
```
White Basket Laundry/
├── hero-bg.jpg       # Background image for hero section
├── Logo.jpg          # Your logo
└── ...other files
```

### Fallback
If no image is provided, the website will display with the original gradient background.

### Testing
After adding the image:
1. Open index.html in a browser
2. Verify the background displays correctly
3. Check text is still readable over the image
4. Test on mobile devices for proper scaling

## Alternative: Multiple Images
You can also create a rotating background by adding multiple images:
- `hero-bg-1.jpg`
- `hero-bg-2.jpg` 
- `hero-bg-3.jpg`
